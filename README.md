# 自律走行システム開発 (ライントレース & 自動追尾)

Arduino（C/C++）を用いた，センサーフィードバック制御による自律走行車両の開発プロジェクトです．「物理的な環境ノイズへの適応」と「応答性の高い制御」をテーマに制作しました．

## ✨ 主な機能・工夫点

*   **物理ノイズへの適応 (PI制御)**
    *   左右のモータの出力個体差や，地面の摩擦変化といった物理的なノイズに対し，PI制御を実装することで直進安定性と滑らかな旋回を両立させました．
*   **環境光への自動最適化**
    *   走行開始時にセンサーの値を読み取り，その場の照明環境に合わせて閾値を自動調整する「自動キャリブレーション機能」を搭載しています．
*   **多機能な走行モード**
    *   **ライントレース:** 高精度なライン判定による高速走行．
    *   **自動追尾:** 超音波センサーを用いた前方物体との距離維持（一定距離を保ちながら追従）．
