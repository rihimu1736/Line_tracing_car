自律走行システム開発 (ライントレース & 自動追尾)Arduino（C/C++）を用いた，センサーフィードバック制御による自律走行車両の開発プロジェクトです．「物理的な環境ノイズへの適応」と「応答性の高い制御」をテーマに制作しました．🎥 デモ映像https://github.com/user-attachments/assets/37930384-a1a3-4b17-b7a7-028e146ac61aPoint: 動画では，急なカーブにおいてもPI制御によるスムーズな減速・旋回を行っている様子が確認できます．🛠 技術スタックカテゴリ技術制御マイコンArduino (C / C++)アルゴリズムPI制御 (比例・積分制御)，自動キャリブレーションハードウェア赤外線フォトリフレクタ，超音波センサー，DCモータ開発環境Arduino IDE✨ 主な機能・工夫点物理ノイズへの適応 (PI制御)左右のモータの出力個体差や，地面の摩擦変化といった物理的なノイズに対し，PI制御を実装することで直進安定性と滑らかな旋回を両立させました．環境光への自動最適化走行開始時にセンサーの値を読み取り，その場の照明環境に合わせて閾値を自動調整する「自動キャリブレーション機能」を搭載しています．多機能な走行モードライントレース: 高精度なライン判定による高速走行．自動追尾: 超音波センサーを用いた前方物体との距離維持（一定距離を保ちながら追従）．📸 外観ギャラリーライントレースカー配線の整理と重心バランスを考慮し，高速走行時の安定性を追求した設計です．自動追尾車前方の物体を検知するためのセンサー配置を最適化しています．
